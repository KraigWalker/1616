# 1616

An ActivityPub server for runners on the fediverse.

## What is this?

This is a server that implements the ActivityPub protocol, which is a decentralized social network protocol. It is designed to be used by runners, but it can be used by anyone.

## How do I use it?

You can use it by signing up at https://1616.run You can also run your own instance of this software, which is what I recommend. You can find instructions for that below.
